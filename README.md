## Install

Clone the repo using

```
$ git clone https://github.com/HarrisonHenri/rentxapi.git
```

```
$ cd rentxapi
```

Install all the required dependencies

```
$ yarn
```

Runs the docker

```
$ docker-compose up
```

Then 

```
$ yarn dev
```
